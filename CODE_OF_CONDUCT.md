# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be welcoming, patient, and respectful. Musicians and programmers of every
  level are invited here, including absolute beginners.
- No harassment, personal attacks, trolling, or publishing others' private
  information.
- Disagree about code and pedagogy as much as you like; criticize ideas, not
  people.

## Enforcement

Report unacceptable behavior privately to the maintainer via GitHub
(@Eipckz). Reports are reviewed and acted on at the maintainer's discretion,
up to and including bans from the project's spaces.
